# Marcello De Feo Portfolio

## Getting Started

```bash
pnpm create next-app . --typescript --eslint --tailwind

pnpm add -D prettier eslint-config-prettier prettier-plugin-tailwindcss eslint-plugin-react eslint-plugin-react-hooks @typescript-eslint/eslint-plugin @typescript-eslint/parser autoprefixer

pnpm add dotenv sanitize-html motion lucide-react

npx shadcn add "https://motion-primitives.com/c/text-effect.json"

touch .prettierrc.json

cd src
mkdir data interfaces lib styles utils

cd app 
mkdir pages

cd ../../

```
